Gerenciamento de Dados

Carregue um backup em JSON (os dados são mesclados com o que já existe), baixe todos os dados do sistema ou envie tudo para o Firebase.

Offline: você pode importar um JSON com saidasAdministrativas e saidasAmbulancias (veja EXPORTAR_SAIDAS_OFFLINE.md na pasta do projeto). Sem internet, os dados ficam no navegador; ao voltar online, use Enviar dados locais para Firebase para atualizar a nuvem.

Usar dados deste computador: ao abrir o SOT pelo GitHub em um computador que já tem dados no navegador, clique aqui para exibir esses dados em vez do Firebase. Usar dados do Firebase: volta a exibir os dados da nuvem.

Esta ação irá apagar TODOS os dados do sistema (local e Firebase) permanentemente.
